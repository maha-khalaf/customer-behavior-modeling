
# Customer Behavior Modeling using Poisson & Negative Binomial Regression

## Overview
This project analyzes customer behavior using count-based statistical models.  
The goal is to predict purchase/exposure frequency and compare model performance.

## Models Used
- Poisson Model
- Negative Binomial Model
- Poisson Regression
- Negative Binomial Regression

## Key Insights
- Negative Binomial models handle variability better than Poisson
- Adding customer features improves model performance
- Demographic variables have limited impact overall

## Tools
- Python
- Pandas
- NumPy
- Jupyter Notebook
